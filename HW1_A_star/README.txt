The graph code is modified based on the code provided by Professor Otte, but much simplied. All the information related to edges has been removed.

The 2d vector operation code is modified based on the 3d vector operation code provided in this Chinese blog: https://blog.csdn.net/zhurui_idea/article/details/24782661.
The line segment and disc intersection check code is modified based on the example code provided in this Chinese blog: https://blog.csdn.net/SongBai1997/article/details/86599879.

All the figures showing the results are located in the test_cases folder.
